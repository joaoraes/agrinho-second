O Projeto Agrinho

Cultivando o conhecimento e a inovação nas próximas gerações.

O Projeto Agrinho é uma iniciativa que visa levar as realidades do campo para o ambiente escolar, instigando o pensamento crítico, a sustentabilidade e o uso de novas tecnologias no manejo agrícola.

Através deste portal, produtores e estudantes podem simular dados em tempo real para otimizar o uso de recursos hídricos e monitorar as condições climáticas essenciais para o sucesso da lavoura.
